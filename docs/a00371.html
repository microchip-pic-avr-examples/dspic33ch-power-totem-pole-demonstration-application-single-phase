static
Parameters
void
Returns
void

if Vout rise up higher than 112% of Vref all switches are switched off that prevents large overshoots when load disappears or load goes down to a low level very quick

if we use Vout_Control.Vout.Filtered as the Vout comparison value, the assessment is only made in the ZC as the Vout average value is determined from ZC to ZC measured. if we use Vout_Control.Vout.Raw as the Vout comparison value, the assessment is made cycle by cycle, every10us. depending on the load jump, this can lead to hickups.
